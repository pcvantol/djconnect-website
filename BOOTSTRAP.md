# DJConnect Website Repository Bootstrap

This repository owns public web presentation, not platform governance. It
adopts AI-Native Engineering Operating System 2.2 from
`pcvantol/djconnect/docs/governance/PLATFORM_ARCHITECT_SYSTEM_INSTRUCTIONS.md`
by reference. Start with `git switch main`, `git pull --ff-only`, current-main,
clean-tree and predecessor verification; read local records, reconcile
`MERGED_UNRECONCILED`, and perform an implementation-reality check. Lifecycle:
`LOCAL_IN_PROGRESS`, `REVIEWABLE_FROZEN`, `MERGED_UNRECONCILED`,
`MERGED_RECONCILED`; cleanup is fail-closed.
