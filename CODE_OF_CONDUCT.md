# Code of Conduct

DJConnect is a small open-source project, and we want every community space
around it to feel respectful, practical and welcoming.

## Our Standards

Examples of behavior that helps the community:

- Be kind, patient and constructive.
- Assume good intent, while still naming problems clearly.
- Give feedback about ideas, code and behavior instead of attacking people.
- Respect different experience levels, backgrounds, languages and time zones.
- Keep discussions focused on improving DJConnect and helping users.
- Avoid sharing private data, credentials, logs with sensitive values or other
  information that should not be public.

Examples of behavior that is not acceptable:

- Harassment, intimidation, threats or sustained personal attacks.
- Discriminatory language or behavior.
- Sexualized language, imagery or attention in project spaces.
- Publishing private information without explicit permission.
- Trolling, repeated disruption or bad-faith participation.
- Using project spaces to promote unrelated products, spam or scams.

## Scope

This code of conduct applies in all DJConnect project spaces, including GitHub
issues, pull requests, discussions, documentation, release channels and direct
project-related communication.

## Reporting

If you see or experience unacceptable behavior, report it by emailing
support@djconnect.dev. Include relevant links, screenshots or context when it is
safe and appropriate to do so. Do not include secrets, tokens, passwords or
private user data in reports.

Reports will be handled with care and discretion. The project maintainer may
remove content, ask participants to change their behavior, temporarily restrict
participation or permanently block someone from project spaces when needed.

## Enforcement Principles

Enforcement should be proportional, consistent and focused on protecting the
community. When possible, we prefer clear guidance and repair. When behavior is
harmful, unsafe or repeated, stronger moderation action may be taken.

## Attribution

This code of conduct is inspired by common open-source community standards and
adapted for the DJConnect project.
