# Website Governance Adoption

This repository adopts AI-Native Engineering Operating System **2.2** from
`pcvantol/djconnect/docs/governance/PLATFORM_ARCHITECT_SYSTEM_INSTRUCTIONS.md`.
There are no local exceptions.

Definition of Done: synchronized clean main, implementation-reality check,
applicable package build/lint/typecheck/link and route-smoke validation,
actual hosting/deployment evidence, rolling records/immutable history and one
reviewable PR.

Release profile: static/web build artifact and actual hosting route smoke.
Docker, Cloudflare Workers and HACS are not universal website targets.
