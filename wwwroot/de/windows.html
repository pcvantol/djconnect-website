Voor vaste bediening

Gebruik je Windows desktop als centrale cockpit voor muziekvragen, output en status.

Ask DJ op desktop

Chat met Ask DJ, gebruik push-to-talk via Home Assistant STT/TTS en laat Home Assistant de server-side history, Music DNA en playback-acties beheren.

Secrets blijven buiten de app

De Windows app bewaart alleen het DJConnect device-token in Windows Credential Manager. Spotify OAuth, Music DNA en backend playback blijven in Home Assistant.

Unsigned validatiebuilds

De Windows downloads zijn voorlopig unsigned builds voor diagnostics en interne validatie. Er is nog geen Store-release, MSIX-package of signed installer.
