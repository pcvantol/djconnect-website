Van kale Pi naar DJConnect

Bereid eerst Raspberry Pi OS voor. De bootstrap hoort bij de Pi-repo en zet de minimale runtime klaar; de publieke app-install staat daarna onder Download.

1

Flash Raspberry Pi OS Lite 64-bit

Gebruik Raspberry Pi Imager en kies een 64-bit Lite image als basis.

2

Configureer eerste boot

Stel in Imager alvast hostname, WiFi, SSH en locale in voordat je de Pi start.

3

Log in via SSH

Start de Pi, log in en installeer Git zodat je de bootstrap uit de Pi-repo kunt ophalen.

4

Haal de Pi-repo op

Gebruik de source checkout alleen voor OS-bootstrap. De app zelf installeer je later uit de publieke release-bundel.

5

Draai de OS-bootstrap

De bootstrap vergroot het bestandssysteem, zet swap, tijdsync, SSH, updates, minimale X11/Qt runtime en HyperPixel setup klaar.

Bootstrap commando 
sudo apt-get update
sudo apt-get install -y git
if [ -d "$HOME/djconnect-pi/.git" ]; then
  cd "$HOME/djconnect-pi"
  git pull --ff-only
else
  git clone https://github.com/pcvantol/djconnect-pi.git "$HOME/djconnect-pi"
  cd "$HOME/djconnect-pi"
fi
sudo ./scripts/bootstrap_raspberry_pi_os.sh
