# DJConnect App 3.2.24

## Ajouts

- Apple Watch propose maintenant Decouvrir avec une liste de recommandations,
  une explication detaillee et Lecture maintenant via le proxy iPhone.
- Les notes What's New sont publiees en JSON en anglais, neerlandais, allemand,
  francais et espagnol.

## Changements

- Le projet Xcode et les schemes partages sont prepares pour les reglages de
  migration Xcode 27.
- Apple Watch regroupe Mood, File d'attente, Playlists et Decouvrir avant Ask DJ
  et les vues d'analyse.
- Track Insight utilise le meme selecteur Mood flottant qu'Ask DJ.
- Les jeux gardent une zone de jeu plus basse sur iPad en paysage afin que les
  boutons sous le jeu restent visibles.

## Corrections

- Le retour haptique fonctionne maintenant aussi en mode demo sur une vraie
  Apple Watch.
