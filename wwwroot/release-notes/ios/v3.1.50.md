# DJConnect App 3.1.50

### Changed

- Ask DJ now opens at the newest message on iPhone, iPad, Mac, and Apple Watch.
- The Ask DJ scroll-to-bottom button now matches the DJConnect gradient style.
- Push registration diagnostics are more detailed while still redacting tokens and secrets.

### Fixed

- Apple Watch pairing now keeps the local Client address on a stable port and advertises mDNS only after the local API listener is ready.
- Apple Watch pairing diagnostics now clearly show whether the Watch has a real WiFi/LAN IPv4 address or is using an iPhone/Bluetooth relay path.
