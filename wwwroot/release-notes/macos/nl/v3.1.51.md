# DJConnect App 3.1.51

## Ask DJ

- Het read-only contract voor technische trackanalyse in Ask DJ is toegevoegd.
- Ask DJ kan nu compacte interpretieve Track Insight tonen, zoals genre,
  energie, productie, luisterpunten en analysebronnen, wanneer Home Assistant
  die teruggeeft.
- Het lege Ask DJ-scherm bevat nu ook een voorbeeldvraag voor technische
  trackanalyse.

---

Platform: macOS

Niet-ondertekende CI-artifacts zijn bedoeld voor diagnostics en interne validatie.
Voor normale macOS-distributie heeft de genotariseerde Developer ID-build de voorkeur wanneer die is bijgevoegd.
