# DJConnect App 3.1.52

## Apple Watch

- Apple Watch-koppeling loopt nu volledig via de gekoppelde iPhone.
- De Watch publiceert geen eigen lokale API en geen Bonjour/mDNS meer.
- Home Assistant ziet nog steeds een normale watchOS-identiteit, maar gebruikt
  de iPhone als blijvende proxy voor lokale callbacks.
- Het Watch-koppelscherm toont alleen nog de koppelcode en iPhone
  companion-status.

Platform: macOS

Niet-ondertekende CI-artifacts zijn bedoeld voor diagnostics en interne validatie.
Voor normale macOS-distributie heeft de genotariseerde Developer ID-build de voorkeur wanneer die is bijgevoegd.
