# DJConnect App 3.2.4

- Music DNA is now first-class on iPhone, Mac, and Apple Watch, with the backend
  profile as the source of truth.
- Ask DJ and Music DNA can show the same Music DNA opt-in prompt on Apple Watch.
- Apple Watch Settings can turn Music DNA off and on again through the iPhone
  proxy while preserving the watchOS identity.
- Music DNA now distinguishes disabled, no profile yet, temporary load failure,
  and demo mode states.
- Apple pairing is now fully inbound-only: iPhone, Mac, and Apple Watch no
  longer expose a local Home Assistant callback API or Bonjour/mDNS discovery.
