# DJConnect Windows 3.1.10

- Adds Ask DJ technical track analysis support for the new contract v2 response shape.
- Shows server-provided analysis sections, optional timeline entries, DJ tips and limitations in a compact read-only card.
- Keeps analysis careful: source/confidence is visible, older interpretive responses still render, and the app does not infer hidden measurements or tips from prose.
- Avoids stale UI by only showing artwork/media and playback buttons when the current response explicitly includes them.

Platform: Windows

Unsigned CI artifacts are intended for diagnostics and internal validation.
Windows artifacts are unsigned and are not a store/MSIX installer yet.
