Geniet van je eigen muziek, met je persoonlijke DJ

Vereisten

Deze checklist voorkomt de meeste installatieproblemen.

  • Home Assistant installatie
  • HACS geïnstalleerd in Home Assistant
  • Spotify Premium account
  • Home Assistant Assist pipeline met werkende STT en TTS
  • DJConnect device of DJConnect iOS/macOS app
  • Home Assistant en DJConnect client op hetzelfde lokale netwerk tijdens pairing
  • Voor Spotify OAuth: een externe HTTPS Home Assistant URL, bij voorkeur Nabu Casa

Waarom Spotify Premium?

DJConnect stuurt playback aan via de Spotify Web API en Spotify playback devices. Daarvoor is Spotify Premium nodig.

Credentials blijven in Home Assistant

DJConnect devices en apps krijgen geen Spotify refresh token of playback credentials.

Assist voor voice

Home Assistant Assist wordt gebruikt voor spraakherkenning en gesproken DJ aankondigingen.

Automatisch

1. Installeer DJConnect in HACS

Gebruik de Home Assistant Community Store deeplink om direct naar de DJConnect repository in HACS te gaan.

Home Assistant Community Store
Handmatig

Voeg de repository zelf toe

  1. Open Home Assistant
  2. Open HACS
  3. Ga naar Integrations
  4. Voeg custom repository https://github.com/pcvantol/djconnect toe
  5. Kies category Integration
  6. Zoek en installeer DJConnect
  7. Herstart Home Assistant
  8. Ga naar Settings -> Devices & services -> Add integration -> DJConnect

2. Koppel Spotify

DJConnect gebruikt Spotify via Home Assistant. Tijdens setup opent Home Assistant de Spotify autorisatiepagina.

Gebruik bij voorkeur je Nabu Casa HTTPS URL als Home Assistant externe URL.
  1. Log in met je Spotify Premium account
  2. Geef toestemming
  3. Je komt terug bij Home Assistant
  4. Daarna beheert Home Assistant de Spotify koppeling

3. Configureer Home Assistant Assist

DJConnect heeft een werkende Assist pipeline nodig voor voice.

STT zet je gesproken muziekvraag om naar tekst. TTS spreekt de DJ aankondiging uit op het DJConnect device of in de app. De DJ aankondiging is instelbaar in de DJConnect options flow.

  1. Open Settings -> Voice assistants
  2. Maak of controleer een Assist pipeline
  3. Kies een STT provider voor spraak naar tekst
  4. Kies een TTS provider voor gesproken DJ aankondigingen
  5. Test de pipeline in Home Assistant
  6. Kies deze pipeline eventueel in de DJConnect options flow

4. Koppel je DJConnect device of app

Kies de route die past bij je client.

ESP device

  1. Zet het DJConnect device aan
  2. Verbind het device met WiFi via setup mode, BLE of captive portal
  3. Open de DJConnect integration setup in Home Assistant
  4. Voer de koppelcode in die op het device-scherm staat
  5. Home Assistant koppelt het device en stuurt een lokaal device token
  6. Wacht tot de statusindicator groen wordt

ESP devices gebruiken lokale communicatie met Home Assistant. De cloud URL wordt niet gebruikt voor device commands. Als mDNS niet werkt, kun je de Client API URL handmatig invullen.

iOS/macOS app

  1. Open de DJConnect app
  2. Ga naar Settings
  3. Kopieer de Client API URL
  4. Open DJConnect setup in Home Assistant
  5. Kies client type iOS app of macOS app
  6. Vul de koppelcode en Client API URL in
  7. Rond pairing af

Voor iOS/macOS is de Client API URL verplicht. De app en Home Assistant moeten elkaar lokaal kunnen bereiken.

5. Controleer of alles werkt

Na pairing kun je deze punten nalopen.

  • DJConnect device/app staat als apparaat in Home Assistant
  • Status is verbonden/gepaired
  • Spotify is gekoppeld
  • Playback proxy kan play/pause doen
  • Voice/PTT start muziek
  • DJ aankondiging verschijnt of wordt uitgesproken
  • Last command, last track en debug attributen worden gevuld
  • Firmware update entity verschijnt alleen voor ESP devices

Problemen oplossen

Geen Spotify playback

Controleer Spotify Premium en of er een actief Spotify afspeelapparaat beschikbaar is.

Geen spraakherkenning

Controleer de Assist pipeline en STT provider.

Geen DJ aankondiging

Controleer TTS provider en DJConnect options.

Pairing lukt niet

Controleer koppelcode, Client API URL en lokaal netwerk.

HACS toont oude versie

Refresh HACS update information en herstart Home Assistant.

Device blijft rood/pending

Controleer of Home Assistant lokaal bereikbaar is vanaf het device of de app.
