DJConnect. Muziekbediening met karakter.

DJConnect brengt spraak, muziekbediening en een persoonlijke DJ-reactie samen. De kern blijft hetzelfde op elk device: vraag muziek, kies waar het speelt en krijg directe feedback terug.

Wat ga je doen?

  1. Integreer DJConnect in Home Assistant
  2. Configureer je Spotify Premium account
  3. Koppel de app of device
Geniet van je eigen muziek, met je persoonlijke DJ

Vereisten

Deze checklist voorkomt de meeste installatieproblemen.

  • Home Assistant installatie
  • HACS geïnstalleerd in Home Assistant
  • Spotify Premium account
  • Home Assistant Assist pipeline met werkende STT en TTS
  • DJConnect device, DJConnect app of Assist Conversation Agent route
  • Home Assistant en DJConnect client op hetzelfde lokale netwerk tijdens pairing, behalve bij Assist Conversation Agent
  • Voor Spotify OAuth: een externe HTTPS Home Assistant URL, bij voorkeur Nabu Casa

Waarom Spotify Premium?

DJConnect stuurt playback aan via de Spotify Web API en Spotify playback devices. Daarvoor is Spotify Premium nodig.

Credentials blijven in Home Assistant

DJConnect devices en apps krijgen geen Spotify refresh token of playback credentials.

Assist voor voice

Home Assistant Assist wordt gebruikt voor spraakherkenning en gesproken DJ aankondigingen.

1. Configureer assist pipeline in Home Assistant

DJConnect gebruikt je Home Assistant voice pipeline voor stembesturing en gesproken DJ aankondigingen.

Hulp bij Home Assistant voice assist
  1. Maak of controleer je voice assist pipeline binnen Home Assistant
  2. Controleer spraakherkenning en gesproken output
  3. Gebruik deze pipeline straks in DJConnect
Automatisch

2. Voeg toe aan Home Assistant

Gebruik onderstaande Home Assistant Community Store knop om direct naar de DJConnect repository in HACS te gaan.

Open DJConnect in HACS
Handmatig

Voeg de repository zelf toe

  1. Open Home Assistant
  2. Open HACS
  3. Ga naar Integrations
  4. Voeg custom repository toe https://github.com/pcvantol/djconnect
  5. Kies category Integration
  6. Zoek en installeer DJConnect
  7. Herstart Home Assistant

3. Configureer

Voeg DJConnect toe als Home Assistant integratie en koppel je Spotify account.

  1. Open Home Assistant Settings -> Devices & services -> Add integration -> DJConnect
  2. Koppel je Spotify Premium account
  3. Configureer je assist pipeline voor stembesturing en pas de stijl van de DJ aankondigingen aan via eigen prompt

4. Koppel app of device

Kies de route die past bij je client.

iOS app

  1. Download iOS app
  2. Open DJConnect app en kopieer de koppelgegevens
  3. Open DJConnect integratie setup in Home Assistant
  4. Kies client type iOS app & plak de koppelgegevens in de Home Assistant integratie
  5. DJConnect app is klaar voor gebruik

macOS app

  1. Download macOS app
  2. Open DJConnect app en kopieer de koppelgegevens
  3. Open DJConnect integratie setup in Home Assistant
  4. Kies client type macOS app & plak de koppelgegevens in de Home Assistant integratie
  5. DJConnect app is klaar voor gebruik

ESP device

  1. Download ESP firmware
  2. Zet het DJConnect device aan en verbind het device met WiFi via captive portal of Home Assistant BLE WiFi provisioning
  3. Open DJConnect integratie setup in Home Assistant
  4. Voer de koppelcode in van het device
  5. Het DJConnect apparaat is klaar voor gebruik

Voice Assistant

  1. Bekijk Voice Assistant uitleg
  2. Kies in de DJConnect setup voor Assist Conversation Agent
  3. Selecteer DJConnect als conversation agent in je Home Assistant Assist configuratie
  4. Gebruik je ESPHome-powered Assist satellite of Home Assistant Voice Preview Edition voor wake word, microfoon en speaker-output
  5. Geen aparte DJConnect hardware-client of koppelcode nodig

Raspberry Pi app

  1. Download Linux app
  2. Installeer de DJConnect Raspberry Pi app via Github
  3. Open DJConnect integratie setup in Home Assistant
  4. Kies client type Raspberry Pi app & plak de koppelgegevens in de Home Assistant integratie
  5. DJConnect Raspberry Pi app is klaar voor gebruik

5. Klaar voor gebruik

Start met een Spotify Connect speaker en bedien daarna je muziek via DJConnect.

  1. Start Spotify Connect op je favoriete speaker
  2. Bedien je muziek via DJConnect
  3. Vraag de volgende artiest aan met je stem
  4. Ontvang persoonlijke DJ aankondigingen in de app of op je device

Problemen oplossen

Geen Spotify playback

Controleer Spotify Premium en of er een actief Spotify afspeelapparaat beschikbaar is. Controleer of de Spotify autorisatie in Home Assistant actief is, of herstel deze.

Geen spraakherkenning

Controleer de Assist pipeline en STT provider.

Geen DJ aankondiging

Controleer TTS provider en DJConnect options.

Pairing lukt niet

Controleer koppelcode, Client API URL en lokaal netwerk.

HACS toont oude versie

Ververs HACS update informatie en download de actuele versie van DJConnect. Herstart Home Assistant.

Device blijft rood/pending

Controleer of Home Assistant lokaal bereikbaar is vanaf de app of device.
