Spraakopdrachten

Wat kun je zeggen?

DJConnect. Muziekbediening met karakter. Vraag via PTT of tekst om muziek; Home Assistant verwerkt je verzoek en DJConnect geeft een persoonlijke DJ aankondiging terug.

Afhandeling
  1. Client neemt spraak op of stuurt tekst
  2. Home Assistant STT maakt tekst
  3. DJConnect leest Assist intentdata
  4. Fallback parsing vult aan wanneer nodig
  5. Spotify start playback via Home Assistant
  6. DJConnect stuurt een DJ aankondiging terug

Artist-first

Generieke muziekvragen worden bewust als artiestverzoek behandeld. “Speel Nirvana” start dus de artiestcontext en kiest niet willekeurig een track of album. Dat houdt spraakverzoeken voorspelbaar.

Nu ondersteund Artist-first bij generieke verzoeken Expliciete mediawoorden sturen de zoekrichting Playback-spraakdekking groeit per release

Intentfamilies

Gebruik gewone zinnen. Voeg alleen woorden zoals nummer, album of playlist toe wanneer je die media specifiek bedoelt.

Artiest

Artiest starten

Zonder expliciet “nummer”, “album” of “playlist” behandelt DJConnect dit als artiestverzoek.

NL

  • speel Nirvana
  • zet Metallica op
  • ik heb zin in Pearl Jam
  • speel maar af Above & Beyond

EN

  • play Nirvana
  • put on Metallica
  • I feel like Fleetwood Mac
  • start Above & Beyond
Track

Nummer / track zoeken

Gebruik woorden zoals nummer, liedje, track of song als je echt een specifieke track bedoelt.

NL

  • speel nummer Black van Pearl Jam
  • start het liedje Everlong
  • zet track Nothing Else Matters op
  • speel het nummer Heroes

EN

  • play song Paranoid Android by Radiohead
  • start track Heroes by David Bowie
  • put on track Everlong
  • play the song Black by Pearl Jam
Album

Album starten

Gebruik album of plaat om album search te forceren.

NL

  • speel album Ten van Pearl Jam
  • start het album Nevermind
  • zet de plaat OK Computer op
  • draai album Rumours

EN

  • play album In Rainbows by Radiohead
  • start the album Nevermind
  • put on the album Blue
  • start album Rumours
Playlist

Playlist starten

Gebruik playlist of afspeellijst als je een playlistnaam bedoelt.

NL

  • speel playlist Roadtrip
  • start mijn playlist Rustig wakker worden
  • zet afspeellijst Dinner Jazz op
  • start afspeellijst Focus

EN

  • play playlist Workout
  • start my playlist Roadtrip
  • put on Chill playlist
  • start Focus playlist
Standaard

Standaard playlist / favorieten

Deze gebruikt de standaard playlist URI die in de DJConnect Home Assistant integration is ingesteld.

NL

  • speel standaard playlist
  • start mijn favorieten
  • zet liked songs op
  • speel mijn favorieten

EN

  • play default playlist
  • start my favorites
  • play liked songs
  • start my default playlist
Uitbreidbaar

Playback control / queue

Playback-bediening loopt via DJConnect backend commands en clientknoppen. Natuurlijke spraakdekking kan per release uitbreiden.

NL

  • pauzeer
  • volgende
  • volume 50 procent
  • wat staat er in de wachtrij?

EN

  • pause
  • next
  • volume 50 percent
  • what is in the queue?
